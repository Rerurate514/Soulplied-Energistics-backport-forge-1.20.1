# Soulplied Enerigistics 1.20.1 Backport
This MOD is https://www.curseforge.com/minecraft/mc-mods/soulplied-energistics backport to 1.20.1

This MOD includes the code from IndustrialForegoingSouls version 1.21 and the code from Soulplied, so it cannot coexist with the existing IndustrialForegoingSouls (1.20.1). Simply adding this MOD will allow you to use IndustrialForegoingSouls and SoulpliedEnergitics.
